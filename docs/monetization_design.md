- **타깃**: 캐주얼~미드코어, 탐험/스토리 선호
- **수익화 우선순위**: 광고 ≈ IAP (균형형)

## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `time_energy` | 시간 에너지 부족 | 에너지 +5 | 5회 |
| `puzzle_hint` | 퍼즐 힌트 필요 | 힌트 1개 | 5회 |
| `explore_reward` | 탐험 완료 | 보상 2배 | 5회 |
| `timeline_unlock` | 타임라인 잠김 | 임시 해금 | 3회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |

### 1.2 전면 광고

- 시대 이동 시 (쿨다운 3분)
- 스토리 몰입 방해 최소화
- 구독자 제외

## 6. 공정성 원칙

- 모든 타임라인 무과금 해금 가능 (시간 투자)
- 스토리 진행에 과금 필수 아님
- 힌트는 편의 요소 (필수 아님)
- SSR 동료는 장기 플레이로 획득 가능
- 가챠 천장: 80회 SSR 확정
