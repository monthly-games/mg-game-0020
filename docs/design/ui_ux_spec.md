# MG-0020: Time Slip UI/UX 스펙

> **게임명**: Time Slip (타임 슬립)
> **장르**: 시간 여행 어드벤처
> **카테고리**: Year 2 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

---

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | 시대, 에너지 |
| 하단 | 34-48dp | 대화 창, 선택지 |
| 좌우 | 16dp | 콘텐츠 여백 |

---

## 3. HUD 레이아웃

```
┌─────────────────────────────────────┐
│  [🕰️ 1920년대]  [⚡3/5]        [⚙️]│  ← 시대/에너지
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐    │
│  │                              │    │
│  │         🏛️ 장면             │    │
│  │                              │    │  ← 스토리 장면
│  │    👤 캐릭터                │    │
│  │                              │    │
│  └─────────────────────────────┘    │
├─────────────────────────────────────┤
│  [캐릭터명]                         │
│  "이 시대에서 단서를 찾아야 해..."   │
│                                     │  ← 대화 창
│  [▶ 다음]                          │
├─────────────────────────────────────┤
│  [선택지 A: 과거로]                 │
│  [선택지 B: 미래로]                 │  ← 분기 선택
│  [선택지 C: 현재에서 조사]          │
└─────────────────────────────────────┘
```

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 대화 진행 | 대화 창 | 탭 |
| 선택지 선택 | 선택지 버튼 | 탭 |
| 스킵 | 화면 우측 | 롱프레스 |
| 백로그 | 화면 상단 | 스와이프 다운 |
