## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 0008 데이터 참조 |
| 3~6주 | 루프 시스템 튜토리얼 강화 |
| 7~10주 | 기본 시간대/지역 밸런스 |
| 11~12주 | 첫 타임라인 이벤트 준비 |

**핵심 과제:**
- 0008 데이터 기반 루프 설계
- 탐험 경로 다양성 확보
- 메타 성장 밸런스

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 시즌별 타임라인 이벤트 1차 |
| 5개월 | 루프별 스페셜 규칙 도입 |
| 6개월 | 새로운 시간대/지역 추가 |
| 7개월 | 시즌별 타임라인 이벤트 2차 |
| 8개월 | 여름 탐험 페스티벌 |

**핵심 과제:**
- 타임라인 이벤트 사이클
- 스페셜 규칙 효과 검증
- 콘텐츠 확장

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 세계관/스토리 이벤트 |
| 10개월 | 0008/0015 세계관 연계 |
| 11개월 | 대규모 타임라인 이벤트 |
| 12개월 | 1주년 탐험 페스티벌 |

**핵심 과제:**
- 세계관 연속성 유지
- 크로스 게임 연계
- 장기 유저 콘텐츠
