## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **타임루프** | 반복되는 루프 속 점진적 발전 | ★★★★★ |
| **메타 성장** | 루프를 반복할수록 강해지는 캐릭터 | ★★★★★ |
| **지식 축적** | 발견한 정보가 다음 루프에서 도움 | ★★★★☆ |
| **분기 탐험** | 선택에 따른 다양한 타임라인 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **더 깊은 진행**: 이전보다 더 먼 층까지 도달
- **영구 성장**: 메타 업그레이드로 강해지는 느낌
- **비밀 발견**: 숨겨진 경로/보물 발견
- **완벽 루프**: 모든 목표 달성 보너스

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 영구 죽음 (루프 내)
├── 제한된 보존 슬롯
├── 랜덤 던전 구조
├── 다양한 타임라인 엔딩
└── 완벽 루프 도전
```

### 1.4 사회적 요소 (Social)
- **일일 도전**: 같은 시드로 경쟁
- **리더보드**: 최대 층수/시간 랭킹
- **시드 공유**: 좋은 시드 공유
- **엔딩 비교**: 달성 엔딩 비교

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (15~30분) - 한 루프
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [탐험 시작] → [방 선택] → [전투/이벤트]          │
│       ↑                            ↓             │
│       └──── [루프 종료] ← [사망/클리어] ←┘       │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **탐험 시작** (10초): 루프 시작, 장비 확인
2. **방 선택** (5초): 다음 방 선택 (전투/상점/이벤트)
3. **전투/이벤트** (2~5분): 방 콘텐츠 진행
4. **사망/클리어** (즉시): HP 0 또는 목표 달성
5. **루프 종료** (30초): 메타 보상, 지식 저장

### 2.2 세션 루프 (30~60분)
```
[접속] → [메타 확인] → [루프 시작] → [탐험] → [루프 종료]
                                          ↓
                              [영구 업그레이드] ← [시간 결정 획득]
```

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [루프 반복] → [시간 결정 축적] → [영구 업그레이드]    │
│         ↓                              ↓             │
│  [지식 축적]                    [새 타임라인 해금]    │
│         ↓                              ↓             │
│  [더 깊은 층 도달] ←──────────────────┘              │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[메인 메뉴] ─────→ [영구 업그레이드]
    │              [지식 DB]
    │              [타임라인 기록]
    │              [설정]
    ↓
[루프 시작] → [층 탐험] → [방 선택] → [콘텐츠 진행]
                              ↓
                    [다음 층] or [루프 종료]
                              ↓
                         [메타 보상]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **리워드 광고** | 40% | 부활, 보상 2배, 추가 슬롯 |
| **타임라인 팩** | 25% | 특별 타임라인 해금 |
| **시즌 패스** | 20% | 시즌 전용 보상 |
| **편의 기능** | 15% | 영구 업그레이드 가속 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 모든 기본 타임라인 플레이 가능
├── 영구 업그레이드 완전 가능
├── 광고 부활 1회/루프
└── 지식 DB 완전 이용

과금 유저 혜택:
├── 광고 제거 ($4.99)
├── 타임라인 팩 ($2.99~$6.99)
├── 크로노 코인 패키지 ($0.99~$19.99)
├── 시즌 패스 ($9.99/시즌)
└── 스타터 팩 ($6.99)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 크로노 코인 (100) | $0.99 | 업그레이드 1회 |
| 중량 크로노 코인 (500) | $3.99 | 20% 보너스 |
| 대량 크로노 코인 (1200) | $7.99 | 40% 보너스 |
| 광고 제거 | $4.99 | 영구 적용 |
| 타임라인 팩 | $2.99~6.99 | 개별 타임라인 |
| 시즌 패스 | $9.99/시즌 | 전용 타임라인 + 보상 |

### 3.4 리텐션 메커니즘
- **일일 도전**: 매일 다른 시드의 특별 루프
- **일일 보상**: 로그인 시간 결정 보너스
- **주간 타임라인**: 한정 타임라인 이벤트
- **시즌**: 3개월 시즌 보상

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
난이도
  ▲
  │                              ╭── 최종 보스
  │                         ╭────╯
  │                    ╭────╯ 보스층
  │               ╭────╯
  │          ╭────╯ 엘리트층
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 층수
    1   3   5   7   9   11  13  15
```

**난이도 공식:**
```
적_체력 = 기본_체력 × (1 + 층수 × 0.25)
보존_슬롯 = 기본_슬롯 + 메타_업그레이드
```

### 4.2 콘텐츠 진행
| 층 | 특징 | 보상 |
|----|------|------|
| 1-3 | 튜토리얼 | 기본 아이템 |
| 4-6 | 성장기 | 중급 아이템, 상점 |
| 7-9 | 분기점 | 타임라인 분기 |
| 10 | 중간 보스 | 타임라인 선택 |
| 11-14 | 고난이도 | 고급 아이템 |
| 15 | 최종 보스 | 엔딩 |
| 16+ | 무한 | 무한 루프 모드 |

### 4.3 밸런싱 공식
```dart
// 메타 업그레이드 효과
double metaBonus(String type, int level) {
  switch (type) {
    case 'startHp': return level * 10; // HP +10/레벨
    case 'startAtk': return level * 2; // ATK +2/레벨
    case 'preserveSlots': return level; // 슬롯 +1/레벨
    default: return 0;
  }
}

// 시간 결정 보상
int timeCrystalReward(int floor, LoopType type) {
  int base = floor * 10;
  double typeBonus = type == LoopType.perfect ? 2.0 : 1.0;
  return (base * typeBonus).toInt();
}

// 지식 DB 힌트
String getHint(KnowledgeDB db, Room room) {
  if (db.hasKnowledge(room)) {
    return room.secretInfo;
  }
  return null;
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 보존 |
|-----------|----------|------|
| 골드 (런 내) | 적 처치, 상자 | 리셋 |
| 시간 결정 | 루프 완료 | 영구 |
| 아이템 | 드롭, 상점 | 슬롯만 |
| 지식 | 발견 | 영구 |
| 크로노 코인 | 구매, 업적 | 영구 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 현재 HP/최대 HP
├── 층수
├── 시간 결정
└── 활성 아이템

레벨 2 (상호작용 시):
├── 방 선택 화면
├── 아이템 상세
├── 적 정보
└── 지식 DB 힌트

레벨 3 (팝업):
├── 루프 결과
├── 영구 업그레이드
├── 타임라인 선택
└── 설정 메뉴
```

### 5.2 네비게이션 흐름
```
[메인 메뉴]
    ├── [새 루프] → [탐험] → [루프 종료]
    ├── [영구 업그레이드] → [업그레이드 목록]
    ├── [지식 DB] → [적/맵/이벤트 정보]
    ├── [타임라인] → [타임라인 기록]
    ├── [일일 도전] → [특별 루프]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 전투 | 전투 이펙트 | 타격음 | 진동 (약) |
| 비밀 발견 | 반짝임 이펙트 | 발견음 | 진동 (중) |
| 사망 | 시간 역행 연출 | 루프 시작음 | 진동 (강) |
| 완벽 루프 | 황금 연출 | 팡파레 | 진동 (강) |
| 엔딩 | 엔딩 컷신 | 엔딩 BGM | 진동 (중) |

### 5.4 접근성
- **턴제**: 시간 제한 없는 전략
- **지식 힌트**: 발견한 정보 자동 표시
- **색맹 모드**: 적/아이템 패턴 추가
- **폰트 크기**: 3단계 조절
- **자동 저장**: 루프 중간 저장
