## 3. 상세 BM 설계

### 3.1 메타 성장 BM

#### 무료 구조
- 루프 종료 시 시간 포인트 획득 (실패해도 지급)
- 발견 진행도에 따른 보너스
- 일일 미션으로 추가 포인트

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 시간 포인트 팩 | 대량 메타 자원 | $2.99~$9.99 |
| 메타 부스트 | 획득량 +50% (7일) | $1.99 |
| 영구 해금 패키지 | 특정 능력 즉시 해금 | $2.99~$4.99 |
| 시작 보너스 강화 | 루프 시작 조건 개선 | $1.99~$3.99 |

### 3.2 탐험가/캐릭터 BM

#### 무료 구조
- 기본 탐험가 3종 무료
- 업적 달성으로 추가 탐험가 2종 해금
- 특정 발견으로 숨겨진 탐험가 해금

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 탐험가 해금 | 개별 탐험가 구매 | $2.99~$4.99 |
| 탐험가 번들 | 탐험가 3종 + 스킨 | $9.99 |
| 탐험가 스킨 | 외형 변경 (성능 동일) | $1.99~$3.99 |
| 시즌 탐험가 | 한정 기간 판매 | $4.99 |

### 3.3 루프 편의 BM

#### 무료 구조
- 일일 루프 횟수 무제한
- 기본 정보 유지
- 타임라인 기본 표시

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 루프 저장 슬롯 | 중간 저장 +2 | $1.99 |
| 빠른 이동 | 발견한 위치 즉시 이동 | 프리미엄 재화 |
| 타임라인 확장 | 상세 기록/분석 | $2.99 |
| 루트 프리셋 | 선호 루트 저장 | $0.99 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 시간 포인트 2배, 힌트, 추가 시간 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $1.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 콘텐츠 무과금 발견 가능
- 탐험가 간 성능 차이는 플레이 스타일 차이
- 메타 성장은 시간이 해결

### 과금 압박 최소화
- 루프 횟수 제한 없음
- 실패해도 시간 포인트 지급
- 탐험 자체가 목표 (압박 없음)
